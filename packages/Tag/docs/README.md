# Tag API

## Tag 属性

| 属性 | 描述 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| type | 类型 | default \| dark \| light \| line | default |
| icon | 设置图标 | string \| slot  | -- |
| color | 字体和icon颜色 | string | #1D2129 |
| borderColor | tag边框颜色 | string | -- |
| backgroundColor | tag背景颜色色 | string | -- |
